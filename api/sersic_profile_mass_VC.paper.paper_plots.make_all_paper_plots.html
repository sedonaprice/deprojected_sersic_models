make_all_paper_plots

sersic_profile_mass_VC.paper.paper_plots.make_all_paper_plots(output_path=None, table_path=None)[source]

Wrapper function to make all plots for the paper.

Saves plots in PDF format.

Parameters

  • output_path (str) – Path to the directory where the plots will be saved.

  • table_path (str) – Path to the directory where the Sersic profile tables are located. Default: system variable SERSIC_PROFILE_MASS_VC_DATADIR, if specified.
